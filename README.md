## 注册
|URL|Header|Method|
| --- | -- | -- |
|/api/signup/| None| POST| 

**URL Params: None** 

**POST Data:**

```
{
  "password" : string,
  "username" : string
}
```
**RETURN Data**
```
{
    "create": int
}
```

**Status Code :**
```
200 // 成功 
401 // 用户名占用
```
